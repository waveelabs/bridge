# BridgePal (Zebegény)

### Pages Created
| Page | File | Description |
| :--- | :--- | :--- |
| **Home / Role Select** | `index.html` | The landing page. Displays the current session status and allows users to enter the app as either the Director (setup) or a Table (score entry). |
| **Director Setup** | `director-setup.html` | The session creation screen. Allows the director to input the session name, date, and configure the number of tables and boards using custom stepper buttons. |
| **Table Entry** | `table.html` | (Skeleton) The screen where players will select their table number before entering board results. |
